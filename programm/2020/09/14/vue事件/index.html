# 需事件处理

在事件绑定后传入多个事件处理程序，内链事件处理程序时，可以出入$event原始DOM事件对象

<button @click="one($event), two($event)">
  Submit
</button>


methods: {
  one(event) {
    // first handler logic...
  },
  two(event) {
    // second handler logic...
  }
}
